Questão 4:
É recomendável utilizar quando se quer executar um determinado trecho de código várias vezes

-> for: é utilizado quando se tem definido a quantidade de vezes que o código será executado, o número de repetições será pré-definido na declaração da estrutura

-> while: é utilizado quando não sabemos quantas vezes o código precisa ser repetido. 

-> do-while: tem quase o mesmo funcionamento que o while, a diferença é que com o uso dele teremos os comandos executados ao menos uma vez.
